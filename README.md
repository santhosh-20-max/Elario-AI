ELARIO AI

📖 Overview

ELARIO AI is an AI-powered learning assistant designed specifically for Electronics and Communication Engineering (ECE) students. It helps users understand core ECE concepts, solve academic doubts, and improve learning through quick, accurate, and interactive responses.

✨ Features

- 🤖 AI-powered ECE question answering
- 📚 Covers core Electronics and Communication Engineering topics
- 💬 Interactive chatbot interface
- ⚡ Fast and user-friendly experience
- 📖 Simplifies complex technical concepts

🛠️ Technologies Used

- Claude AI
- DeepSeek API
- Amazon Q
- HTML
- CSS
- JavaScript

🚀 Getting Started

1. Clone this repository.
2. Install the required dependencies.
3. Configure your API keys.
4. Run the application.

📌 Future Enhancements
